Expense Log System
Project Overview

A web application that allows users to add, view, and delete expenses. Each expense includes a type, amount, and optional notes, with a running total that updates automatically. This project was refactored from a previous Favorite Links application to serve a new purpose.

Technologies Used

HTML, CSS, JavaScript, Node.js, Express, PostgreSQL (Neon), Git, GitHub

Installation / Setup

Clone the repository

Navigate into the project directory

Run npm install

Create a .env file and add your PostgreSQL connection string:

DATABASE_URL=your_neon_postgres_url


Start the application with npm start
